Configuration Examples

snowman.yaml

Here is a value/key pair from your snowman.yamlfile.

a_config_key: {{ config.Metadata.a_config_key }}

SPARQL

If it suits your case you can define configuration using SPARQL's BIND and VALUES keywords! The following value and list comes from a query file.

{{ index (index . 0).fancy_value }} {{ range . }} {{ .uri }} | {{ .name }} {{ end }}

Environment variables

You can also set configuration values using environment variables! In case you have PATH set it will render below.

{{ env "PATH" }}

Current View Configuration

You can also get configuration values related to the current view.

{{ current_view }} QueryFile: {{ current_view.QueryFile }}
