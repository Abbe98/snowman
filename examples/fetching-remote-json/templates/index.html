Snowman fetching and parsing remote JSON

First lets just use get_remote to fetch a JSON file as a string:

{{ get_remote "https://fornpunkt.se/lamning/lNJVbNa.geojson" }}

You can fetch any resource that way, but now let's use from_json to parse it and extract the title-property.

{{ (from_json (get_remote "https://fornpunkt.se/lamning/lNJVbNa.geojson")).properties.title }}
